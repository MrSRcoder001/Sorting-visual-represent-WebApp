The application generates a random array of numbers.

Each array element is displayed as a vertical bar.

When a sorting algorithm is selected, the bars animate to show comparisons and swaps.

Colors indicate different operations such as comparison, swap, and sorted state.

The process continues until the array is fully sorted.
